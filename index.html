Introduction

10secondsofcode

10secondsofcode is a easy way to understand the reactjs snippets in 10 seconds.

Website : https://10secondsofcode.github.io/10secondsofcode/

Getting Started

git clone https://github.com/10secondsofcode/10secondsofcode.git
npm install
npm start

Built With

  • ReactJs
  • BootStrap4
